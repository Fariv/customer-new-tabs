# Customer Edit View Prototype

This is a mockup for a **customers edit view** interface designed with two additional tabs:

## Tabs

### 1. **Projects Tab**
Displays a list of projects associated with the customer. Features include:
- Project status tracking
- Customer and location information
- Project details and descriptions
- Creation dates and total sales amounts
- Project manager assignment
- Search and field selection capabilities
- Pagination for handling multiple records

### 2. **Message View Tab**
A messaging interface for communication related to projects. Features include:
- User messages and communication threads
- Project status indicators
- Message titles and content
- Recipient tracking
- Entity and message associations
- Search functionality
- Pagination support

## Design Details

- **Framework**: HTML5 with inline CSS
- **Color Scheme**: Professional business palette with teal, gold, and blue accents
- **Responsive**: Adapts to smaller screens (breakpoint at 1250px)
- **Interactive Elements**: Tab navigation, modal dialogs for field selection, search functionality

## Functionality

- **Tab Navigation**: Click on any tab to switch between views
- **Field Selection Modal**: Drag fields between available and selected columns to customize table columns
- **Search**: Quick search functionality for filtering records
- **Pagination**: Navigate through multiple pages of results
